## 📝 Description
This is a simple GUI-based number-guessing game built with Java Swing.  
The program generates a random number between **1 and 100**, and the user attempts to guess it using a text input field.

After each guess, the game provides feedback:
- 🔼 Too high  
- 🔽 Too low  
- ✅ Correct  

The application also keeps track of the total number of guesses. A reset button allows the user to start a new game at any time.

## ⚙️ Features
- 🎲 Random number generation (1–100)
- ⌨️ User input via `JTextField`
- 📢 Real-time feedback using `JLabel`
- 🔢 Guess counter display
- 🔄 Reset button to restart the game
